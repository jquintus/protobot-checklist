# checklist

> A GitHub App built with [Probot](https://github.com/probot/probot) that Only allows a Pull Request to be merged when all items in the PR&#x27;s checklist have been ticked off.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
